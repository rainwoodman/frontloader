class frontloader.tinydb.storages.Storage¶

Bases: object

The abstract base class for all Storages.

A Storage (de)serializes the current state of the database and stores it in some place (memory, file on disk, ...).

close()¶

Optional: Close open file handles, etc.

read()¶

Read the last stored state.

Any kind of deserialization should go here. Raise ValueError here to indicate that the storage is empty.

Return type:dict
write(data)¶

Write the current state of the database to the storage.

Any kind of serialization should go here.

Parameters:data (dict) – The current state of the database.
